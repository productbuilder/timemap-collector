Tools

Open Collections Protocol is the shared model for publishing and discovering collections. These tools make that model practical: helping teams publish, register, index, and browse collections with ordinary web URLs.

Core tools

Collection Manager

Create, update, and publish collection manifests and item records without hand-editing JSON.

Purpose: publishing workflows for collection owners and editors.

Best for: teams preparing and releasing collections.

Available now
Open Collection Manager Publish guide

Collection Browser

Explore published collections in a friendly interface for websites, portals, and discovery pages.

Purpose: practical viewing and navigation for published content.

Best for: readers, researchers, and public catalog experiences.

Available now
Open Collection Browser View demo

Collection Registry

Keeps track of known collection endpoints so systems can discover what has been published.

Purpose: shared discovery and registration workflows.

Best for: aggregators, ecosystem hubs, and directory services.

Planned service Documentation available
Registry docs

Collection Indexer

Processes collection manifests and items into search and discovery outputs for portals and APIs.

Purpose: fast discovery, filtering, and search readiness.

Best for: search experiences and multi-collection catalogs.

Planned service Documentation available
Indexer docs

Components and embedding

Collection Manager and Collection Browser are also available as embeddable web components. You can place them inside websites, portals, apps, or existing collection systems. 

<script type="module" src="/src/apps/collector/src/index.js"></script>
<collection-manager></collection-manager>

This loads the Collection Manager component directly and renders it inline on your page. See full options in components and embedding.

How the tools fit together

  • Publish with Collection Manager.
  • Register endpoints in Collection Registry.
  • Process content with Collection Indexer.
  • Browse and reuse with Collection Browser.
