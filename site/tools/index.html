Tools

Open Collections Protocol provides a shared web surface. The tools below help people publish, register, index, and explore collections using that shared model.

Publishing and viewing tools

Collection Manager

Editing and publishing tool for creating collections, organizing items, and publishing collection.json and related assets.

Collection Browser

Viewing and browsing tool for reading published collections. Useful for public-facing sites, portals, and embeddable collection displays.

Both Collection Manager and Collection Browser can run as standalone apps or as embeddable components.

Discovery and processing tools

Collection Registry

Public-facing registry for known collections, publishers, and registry links. Registrator is the internal architecture/module term.

Collection Indexer

Processing/indexing layer that reads manifests and item records to build discovery, search, and aggregation views.

Registry and Indexer can be deployed together early on, while remaining distinct roles.

Exploratory presentation

TimeMap

Exploratory presentation layer for navigating collections through time and context. It complements Collection Browser and can draw on indexed and curated datasets.
