Tools

Open Collections includes practical tools for creating and browsing collections. Start with the two tools most teams need first.

Main tools

Collection Manager

Use Collection Manager to organize files, add collection details, and save your updates.

It runs in the browser and can also be used in a desktop-style shell for local-folder workflows.

Available now
Open Collection Manager Get started guide

Collection Browser

Use Collection Browser to open and explore collections in a friendly viewing experience.

It works well for websites, public pages, and internal browsing workflows.

Available now
