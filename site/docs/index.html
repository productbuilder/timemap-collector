Documentation

This documentation helps you understand, publish, and integrate collections with Open Collections Protocol.

Some pages are written for collection owners and publishers. Others go deeper into technical implementation, components, and integration.

Start here

If you are new to the project, start with these pages:

  • Protocol — understand the core publishing model
  • Tools — see which tools help create, publish, and view collections
  • Storage Providers — understand practical hosting options
  • Demo — see the tools and components in action

Publishing a collection

If your goal is to get a collection online, the most important ideas are:

  • create a collection manifest
  • publish items and media at stable URLs
  • choose a storage provider that fits your needs
  • optionally publish a domain discovery file

Tools and components

Open Collections Protocol is supported by tools such as:

  • Collection Manager
  • Collection Browser
  • Indexer
  • Registrator
  • TimeMap

Collection Manager and Collection Browser can also be used as embeddable components inside websites, portals, and other applications.
