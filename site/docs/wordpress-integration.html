WordPress integration

WordPress can be a practical adoption path. The core public contract remains the protocol outputs and stable URLs.

Where WordPress helps

  • familiar CMS workflows for non-technical teams
  • plugin-based integration into institutional websites
  • fast path to publishing manifests and item pages

Protocol outputs to preserve

  • collection.json manifest URL
  • stable item and media URLs
  • DCD file at /.well-known/collections.json when applicable
  • optional registry export for registry participation
