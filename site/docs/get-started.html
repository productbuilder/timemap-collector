Get started (technical companion)

Use this with the public Get started page when your technical team needs deeper implementation detail.

Collection outputs

  • collection.json at a stable URL
  • stable item and media URLs
  • optional domain discovery file at /.well-known/collections.json

Practical flow

  1. Prepare folder structure and file naming.
  2. Open Collection Manager and generate/update collection data.
  3. Validate links and metadata.
  4. Host files and verify public URLs.
