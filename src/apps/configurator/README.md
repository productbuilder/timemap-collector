# Configurator App

Local-first package manager/editor for configurator JSON files.

Open this app at:

`/src/apps/configurator/`

Current first version supports:

- open existing JSON package files
- section navigation from top-level arrays/objects
- center browse panel (rows/cards where useful)
- right inspector with generic editable fields
- save back to the same file when File System Access API is available
- save-as fallback to file picker or download
