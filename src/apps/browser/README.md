# Browser App

TimeMap Browser is the read-only collection browsing app.

## Scope

- Load collection manifests
- Browse item cards
- View media in a larger viewer
- View metadata in read-only mode

## Entry

- App host page: `/src/apps/browser/`
- Web component script: `/src/apps/browser/src/index.js`

## Notes

- Built with vanilla JS + Web Components.
- Uses shared helpers from `/src/library/core/`.
